Workflow Notes

                    
How the web app and bot fit together

                

                

                    

                        
What does the site do?

                        
It showcases DC-Fold as a deployable Discord infrastructure bot and includes the JSON blueprint builder as a secondary utility in the site menu.

                    

                    

                        
What does the bot do?

                        
It reads template JSON, creates roles and channels, applies staff visibility rules, sets slowmode, and seeds starter embeds in text channels.

                    

                    

                        
Where does GitHub Pages deploy from?

                        
The workflow publishes the static contents of web-app, so the site can live independently from the bot runtime.

                    

                    

                        
Can I use the builder for custom deploys?

                        
Yes. Export or copy the generated JSON, then use the bot's custom deploy path with an uploaded file or pasted payload.
